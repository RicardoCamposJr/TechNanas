# TechNanas
</div>


## Sobre o projeto:

### Para o produtor:
Um sistema de EAD totalmente funcional e interativo, para produtores de conteúdo tech. No sistema é possível cadastrar cursos em diferentes categorias, bem como, cadastrar as aulas de cada um deles. É possível também criar categorias, e pôr em evidência algum curso de preferência.

### Para o usuário:
O sistema permite que o usuário se capacite profissionalmente no mundo da programação através de vídeo aulas de diversos cursos cadastrados na plataforma.

## Instalação:
### Clone o repositório do servidor da aplicação e rode-o:
```
$ git clone https://github.com/RicardoCamposJr/ead.git
$ npm i
$ npm run dev
```
### Após isso, clone o repositório da aplicação atual e rode-o:
```
$ git clone https://github.com/RicardoCamposJr/TechNanas.git
$ npm i
$ npm run dev
```
#### OBS: É necessário ter uma conexão com o banco de dados local da sua máquina, o projeto não teve o deploy ainda.
